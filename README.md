# 🎬 Movie Catalog

Projeto web desenvolvido para praticar **JavaScript, manipulação de DOM e consumo de APIs**, permitindo buscar filmes e salvá-los como favoritos.

## 📌 Sobre o projeto

O **Movie Catalog** é uma aplicação web onde o usuário pode pesquisar filmes através de uma API externa e organizar seus favoritos.
O projeto também possui um sistema simples de **login e cadastro**, com dados armazenados no navegador.

Este projeto foi desenvolvido com o objetivo de **praticar conceitos de desenvolvimento web e organização de código**.

---

## 🚀 Funcionalidades

* 🔎 Buscar filmes pelo nome
* ⭐ Adicionar filmes aos favoritos
* ❤️ Visualizar lista de filmes favoritos
* 🔐 Sistema de login
* 📝 Cadastro de usuário
* 💾 Armazenamento de dados usando LocalStorage

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* API de filmes (OMDb API)
* LocalStorage
